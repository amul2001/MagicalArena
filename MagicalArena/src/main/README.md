# Magical Arena

This project simulates a battle between two players in a magical arena.

## How to Run

1. Clone the repository.
2. Navigate to the project directory.
3. Compile the project:
    ```
    mvn compile
    ```
4. Run the project:
    ```
    mvn exec:java -Dexec.mainClass="com.arena.Main"
    ```
